Setting Up subte.cc

Updating a domain manually

Domains can be updated using a simple GET request with your user token as the header to the following API:

https://subte.cc/up?domains=YOURDOMAINNAMEHERE&ip=YOURIPHERE

Example with curl

curl -H "X-API-Key: YOURTOKENHERE" https://subte.cc/up?domains=YOURDOMAINNAMEHERE&ip=YOURIPHERE

If no IP is specified in the request, the IP from where the request is made will be used.

Updating a domain using cron

This method works in Unix-based systems with curl and cron installed.

1. Open the crontab file with:

$ crontab -e

2. Add the cron job to the crontab file and save:

0 * * * * curl -H "X-Api-Key: YOURTOKENHERE" https://subte.cc/up?domains=YOURDOMAINNAMEHERE&ip=YOURIPHERE

(Updates every hour)
